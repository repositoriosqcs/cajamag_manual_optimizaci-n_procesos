    7. _incMesActualvsAnteriorAfilMesAnterior

    Descripción: Comparación entre el periodo de estudio y el anterior para individuos activos.

    8. _incMesActualvsAnteriorAfilMesAnterior(Año)

    51. _ValidacionCumpleEncuestas

    Descripción: Evalúa si el número de encuestas aplicadas cumple con lo establecido por la Super.

VAR maxbene = SELECTEDVALUE(MetasEncuestas[Max Beneficiarios])
VAR minen = SELECTEDVALUE(MetasEncuestas[Min Encuestas])

    1,
    0
)

    IF(numEncu < minen, "No Cumple", "Cumple")

    72. _satisfaccionMesActualvsAnterior

    Descripción: Variación de la calificación por pregunta de encuesta, del periodo de estudio versus el periodo anterior.

    
VAR periodoActual = SELECTEDVALUE(Periodos[periodo])
VAR periodoAnterior = CALCULATE(MAX(Periodos[periodo]), ALL(Periodos), Periodos[periodo] < periodoActual)
VAR encAnt = CALCULATE([_Calificacion], ALL(Periodos), Periodos[periodo] = periodoAnterior)
RETURN IF(ISBLANK(encAnt), BLANK(), ([_Calificacion] - encAnt) / encAnt)
    73. _satisfaccionMesActualvsAnterior(Año)

    Descripción: Variación de la calificación por pregunta de encuesta, del periodo de estudio versus el mismo periodo año anterior.

    
VAR periodoActual = SELECTEDVALUE(Periodos[periodo])
VAR periodoAnoAnterior = (LEFT(periodoActual, 4) - 1) * 100 + RIGHT(periodoActual, 2)
VAR afilAnt = CALCULATE([_Calificacion], ALL(Periodos), VALUE(Periodos[periodo]) = periodoAnoAnterior)
RETURN IF(ISBLANK(afilAnt), BLANK(), ([_Calificacion] - afilAnt) / afilAnt)
    74. _TamañoMuestra

    Descripción: Muestreo sugerido de acuerdo a la población afiliada en el periodo con un nivel de confianza del 90%.

    
VAR Z = 1.645
VAR p = 0.5
VAR E = 0.1
VAR Poblacion_Total = CALCULATE(DISTINCTCOUNT(Historico[cod]), Historico[seAfilio] = 1)
RETURN (Z^2 * p * (1 - p) * Poblacion_Total) / ((E^2 * (Poblacion_Total - 1)) + (Z^2 * p * (1 - p)))
