    Indicadores y Medidas Calculadas¶

    A continuación, se presentan los indicadores y medidas calculadas, junto con su descripción, fórmula y meta (si aplica).

    1. _muestra¶

    Descripción: Cálculo del total de muestras a revisar en el proceso de grabación.
    Fórmula: 

    IFERROR( VAR formula = (SUM( 'ErroresGrabación'[Muestra a revisar] ) / DISTINCTCOUNT ('ErroresGrabación'[Error] )) 
RETURN IF(ISBLANK(formula), "-", formula), "-")

    2. _#ErroresAfiliaciones¶

    Descripción: Total de errores de grabación encontrados en la muestra a verificar.
    Fórmula: 

    IF (ISBLANK( SUM('ErroresGrabación'[NumErrores] ) ), "-", SUM('ErroresGrabación'[NumErrores]))

    3. _%desafiliaciones¶

    Descripción: Participación porcentual de desafiliación por motivo.
    Fórmula: 

    SUM(Historico[seRetiro]) / CALCULATE( SUM(Historico[seRetiro]), REMOVEFILTERS(EstadosDim[detest]))

    4. _%ErroresAfiliaciones¶

    Descripción: Porcentaje de errores de grabación encontrados en relación con la muestra validada.
    Fórmula: 

    IFERROR( VAR formula = (SUM( 'ErroresGrabación'[NumErrores] ) / (SUM( 'ErroresGrabación'[Muestra a revisar] ) / DISTINCTCOUNT ('ErroresGrabación'[Error] ))) 
RETURN IF(ISBLANK(formula), "-", formula), "-")

    5. _%ErroresAfilPoblacion¶

    Descripción: Proyección de errores potenciales en formularios analizados.
    Fórmula: 

    IFERROR( VAR formula = ( VAR pob_afiliada = CALCULATE( sum(Historico[seAfilio] ), OR( Historico[Tipo] = "Trabajador", Historico[Tipo] = "Beneficiario")) 
VAR total_errores = SUM('ErroresGrabación'[NumErrores] ) 
VAR muestra = SUM( 'ErroresGrabación'[Muestra a revisar] ) / DISTINCTCOUNT ('ErroresGrabación'[Error] ) 
RETURN IF (ISBLANK( ( pob_afiliada * total_errores ) / muestra), "-", (( pob_afiliada * total_errores ) / muestra))) 
RETURN IF(ISBLANK(formula), "-", formula), "-")

    6. _%ParDesafiliacion¶

    Descripción: Participación de la desafiliación en relación con el total de afiliaciones.
    Fórmula: 

    CALCULATE( DISTINCTCOUNT(Historico[cod]) , Historico[seRetiro] = 1 ) / DISTINCTCOUNT(Historico[cod])

    7. _incMesActualvsAnteriorAfilMesAnterior¶

    Descripción: Comparación entre el periodo de estudio y el anterior para individuos activos.
    Fórmula: 

    IFERROR( VAR formula = ( VAR periodoActual = MAX( Periodos[periodo] ) 
VAR periodoAnterior = CALCULATE( MAX( Periodos[periodo] ), ALL(Periodos), Periodos[periodo] < periodoActual ) 
VAR afilAnt = CALCULATE( DISTINCTCOUNT( Historico[cod] ) - CALCULATE( SUM( Historico[seAfilio])), ALL(Periodos) , Periodos[periodo] = periodoAnterior ) 
RETURN (DISTINCTCOUNT( Historico[cod] ) - CALCULATE( SUM( Historico[seAfilio])) - afilAnt ) / afilAnt) 
RETURN IF(ISBLANK(formula), "-", formula), "-")

    8. _incMesActualvsAnteriorAfilMesAnterior(Año)¶

    Descripción: Comparación del resultado entre el periodo de estudio y el mismo periodo año anterior para los individuos activos en el mes anterior.
    Fórmula:

    9. _incMesActualvsAnteriorAfilNuevas¶

    Descripción: Comparación del resultado entre el periodo de estudio y el periodo anterior para los individuos que se afiliaron en el periodo de estudio.
    Fórmula: 

    IFERROR( VAR formula = (
    VAR periodoActual = MAX( Periodos[periodo] )
    VAR periodoAnterior = CALCULATE(
        MAX( Periodos[periodo] ),
        ALL(Periodos),
        Periodos[periodo] < periodoActual
    )
    VAR afilAnt = CALCULATE(
        SUM( Historico[seAfilio] ),
        ALL(Periodos),
        Periodos[periodo] = periodoAnterior
    )
    RETURN (SUM( Historico[seAfilio] ) - afilAnt ) / afilAnt

RETURN IF(ISBLANK(formula), "-", formula), "-")

    84. _Recuperado_presunto_formato¶

    Descripción: Para una empresa en particular, si está en mora y en el mes siguiente no lo está, se toma como recuperado lo que debía en el mes anterior.
    Fórmula: 

    IFERROR(
    VAR perac = SELECTEDVALUE(Periodos[periodo])
    VAR actYear = LEFT(perac, 4)
    VAR actMonth = RIGHT(perac, 2)
    VAR antPer = CONVERT(
        IF(actMonth = "01", (actYear - 1) * 100 + 12, (actYear) * 100 + (actMonth - 1)),
        STRING
    )
    RETURN CALCULATE(
        SUM(HistoricoMoraEmp[valormor]),
        ALL(Periodos),
        Periodos[periodo] = antPer,
        HistoricoMoraEmp[Ultimo] = 1
    )
    RETURN IF(ISBLANK(formula), "-", formula),
    "-"
)
